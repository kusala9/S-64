NON OFFICIAL BASE CELL

This directory contains a single base cell.

It is a modified copy of gb5x01ne.000 from the goodb1
directory. The modifications are described below:

The producer code has been changed from GB to 1B.

The agency code (AGEN) has been modified from 540
to 65535 as specified in S-57 clauses 4.3.1 and 2.1.
